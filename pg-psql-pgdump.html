Psql 和 PgDump 的使用

Postgresql的备份和迁移

准备

在新库中创建相同的角色(用户)

create user some_user with password 'password'; create database some_db owner some_user; grant all privileges on database some_db to some_user;

旧库备份

pg_dump -U some_user -d some_db > some_db.sql

新库导入

psql -U some_user -d some_db < some_db.sql

Psql的使用

登录

psql -U some_user -d some_db

切换库

\c some_db

查看所有表

\dt
